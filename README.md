# FeedbackPulse AI

SaaS-сервис для AI-анализа клиентских отзывов. Автоматически определяет тональность, категоризирует темы, выявляет аномалии и формирует actionable-рекомендации.

## Возможности

- **Импорт данных** — Drag-and-drop загрузка CSV и Excel файлов (до 5MB)
- **Умный маппинг** — Автоматическое определение колонок с текстом отзывов и рейтингом
- **AI-анализ** — Определение тональности (positive/negative/neutral), категоризация тем, обнаружение аномалий
- **Визуализация** — Интерактивные графики: круговая диаграмма тональности, столбчатая диаграмма тем
- **Рекомендации** — Автоматически сгенерированные actionable-советы на основе анализа
- **Экспорт** — Выгрузка полного отчета в текстовый файл

## Технологии

- **Frontend**: Next.js 16, React 19, TypeScript, Tailwind CSS v4
- **UI**: shadcn/ui, Recharts
- **AI**: Vercel AI SDK 6, OpenAI GPT-4o-mini
- **Парсинг**: PapaParse (CSV), SheetJS (Excel)

## Установка

```bash
# Клонирование репозитория
git clone <repository-url>
cd feedbackpulse-ai

# Установка зависимостей
pnpm install

# Настройка переменных окружения
cp .env.example .env.local
# Добавьте OPENAI_API_KEY в .env.local

# Запуск dev-сервера
pnpm dev
```

## Переменные окружения

| Переменная | Описание |
|------------|----------|
| `OPENAI_API_KEY` | API ключ OpenAI для AI-анализа |

## Использование

1. Загрузите файл с отзывами (CSV или Excel)
2. Проверьте автоматический маппинг колонок или настройте вручную
3. Нажмите "Запустить анализ"
4. Изучите результаты на дашборде
5. Экспортируйте отчет при необходимости
